**多目标物品自动计数与位置标注系统**

1、项目简介

本系统是《机器视觉与图像处理》课程大作业（方向二）的完整实现。

针对散落的硬币、瓶盖、零件等图像，通过图像预处理、二值化、形态学运算和连通域分析，自动统计画面内目标总数、框选每个目标并输出坐标位置，模拟工业视觉批量计数与定位场景。

2、主要功能

\- 弹窗选择本地图片，操作简单

\- 自动灰度化 + Otsu自适应二值化

\- 形态学开闭运算去噪与修补

\- 连通域标记与面积阈值过滤

\- 在原图上红色矩形框选 + 黄色编号标注

\- 绿色十字标记质心

\- 命令窗口输出每个目标的编号、质心坐标、宽度和高度

\- 绘制面积分布直方图，辅助参数调优

5、运行步骤

（1） 确保MATLAB R2016a已安装且包含Image Processing Toolbox。

（2）将本仓库所有文件下载到本地。

（3）在MATLAB中打开 main.m文件。

（4）直接点击“运行”按钮运行程序。

（5）在弹出的文件选择对话框中，选择 文件夹中的任意一张测试图片。

（6）等待约0.5秒，MATLAB将自动生成结果窗口，包含：原始图像、二值化处理结果、框选标注后的计数结果（显示总数）、面积分布直方图。

（7）查看MATLAB命令窗口，程序已输出每个目标的坐标信息表。

6、实验结果与结论

在目标相互分离、背景对比度良好的条件下，本系统的计数准确率可达 100%。系统能够准确输出每个目标的质心坐标和边界尺寸，验证了基于连通域分析的图像处理方案在结构化工业场景中的有效性与工程实用价值。
